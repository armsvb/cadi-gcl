Cadi ADC test
========

This test displays 4 adc values read from JDR registers.
Useful for checking analog buttons functionality

Данный тест отображает 4 значения с аналоговых входов (4 ресгистра JDR).
Минимальное требование - работающий дисплей для отображения текущих значений АЦП.
Тест полезен, например, для проверки правильности подсоединения аналоговых кнопок
